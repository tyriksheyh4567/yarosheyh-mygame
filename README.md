# Своя Игра / Nuxt UI

## Setup

Make sure to install the dependencies:

```bash
pnpm install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
pnpm run dev
```

## Production

Build the application for production:

```bash
pnpm run build
```

Locally preview production build:

```bash
pnpx nuxthub preview
```

Deploy to NuxtHub:
```bash
pnpx nuxthub deploy
```
